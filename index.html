    LikePy

    This script traces the likes on posts of a page, by your team mates / a list of friends. You must be the admin of the page to use this scripts. It's based on Facebook-sdk for python [https://facebook-sdk.readthedocs.org/]

    Requirements

    • Python

    • pip

      sudo easy_install pip

    • virtualenv

      sudo pip install virtualenv

    • facebook-sdk

      sudo pip install facebook-sdk
